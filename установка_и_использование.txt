Установка.
	Windows:

		python -m pip install aiogram pillow pycoingecko

	Linux:

		python3 -m pip install aiogram pillow pycoingecko

Подготовка.

Для начала откройте config.json
Все полученные далее значения необходимо внести в него.

Если вы хотите создать бота вам понадобится токен.
Обратитесь к @BotFather в телеграме для его получения:
	1. Используйте команду/newbot
	2. Выберите ему имя и юзернейм.
Готово. Теперь вставьте в графе bot_token этот токен.

Если вы создавали бота, то вам понадобится ввести и ваш user_id,
	его можно получить у @userinfobot.
Вставьте его в графе user_id.

В stickerset_id вы вводете краткое имя стикерпака, которое нужно для того, чтобы добалять пак по ссылке:
	tg://addstickers?set=sticker_id_by_имя_бота

В stickerset_title вы вводите название стикерпака, которое будет показываться в телеграме.

blacklist — список монет, которые не нужно показывать.

amount_of_stickers — количество стикеров, которые нужно создавать для стикерпака.

arrow_red, arrow_green — пути к файлам, где находятся изображения стрелок.

update_time — время обновления в минутах

default_font — шрифт, который используется для отображения текста, если не было указано иначе.

default_font_size — шрифт, который используется для отображения текста, если не было указано иначе.


default_color — цвет, который используется для отображения текста, если не было указано иначе.

Далее, у каждого объекта стикера есть несколько возможных параметров:
	coordinates — обязательный параметр. Указывает на координаты, где отрисовывать объект.
	enabled — обязательный параметр. Указывает, включён ли параметр.
	centered — необязательный параметр. Значение по умолчанию — true.
	font_size — размер шрифта
	color — цвет текста
	font — имя файла, в котором находится шрифт.

Возможные объекты:
	name — название монеты, например, "Bitcoin".
	rate — соотношение валют, например, "BTC/USDT".
	price — цена на монету.
	previous_price — цена на монету.
	market_cap — market cap, сведён к формату 1B, 1.23B, 1.6T, и т.д.
	volume — объём торгов за последние 24 часа, сведён к формату 1B, 1.23B, 1.6T, и т.д.
	change_24h — изменение цены в процентах, ограничено 2 символами после запятой.
	coin_icon — иконка монеты.


Запуск скрипта.

	Windows:

		python main.py

	Linux:

		python3 main.py
